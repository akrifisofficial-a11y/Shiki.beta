Для плеера Shikimori • Только через VK Player

ВАЖНО: Работает только с видео, загруженными через VK Player. Другие источники не поддерживаются.
Формат: https://vk.com/video-XXXXXXX_XXXXXXXX

Как это работает?

  1. Загрузите видео на VK Видео
  2. Скопируйте ссылку на видео
  3. Заполните форму
  4. Заявка придет в Telegram-чат модераторам
  5. После проверки серия появится в плеере
